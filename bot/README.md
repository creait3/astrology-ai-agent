# Bot

This directory will contain code for the conversational bot and personalized horoscope service.

Recommended tasks:

- Develop a backend service that uses vector search (Pinecone) to answer user queries in the style of their favorite influencer, leveraging the embeddings generated in the embedding module.
- Integrate with the Lovable platform to embed a chat widget on the website for natural language interactions.
- Implement a workflow to generate personalized horoscopes and deliver them via WhatsApp using the Charles API; ensure user consent and authentication.
- Capture conversation logs and user feedback to update relevance scores and improve the model over time.
